📄 Estrutura do Documento
TagO que fazExemploObs
<!DOCTYPE html>meta Declara que o documento é HTML5. Deve ser a primeira linha do arquivo. <!DOCTYPE html> Obrigatório. Não é uma tag HTML de verdade, é uma instrução.
<html>bloco Elemento raiz. Envolve todo o conteúdo da página. <html lang="pt-BR">...</html> Use o atributo lang para acessibilidade e SEO.
<head>meta Contém metadados da página: título, CSS, scripts, charset. Não é visível. <head>...</head> Tudo aqui é lido pelo navegador mas não aparece na tela.
<body>bloco Contém todo o conteúdo visível da página. <body>...</body> Só pode haver um <body> por página.
<title>meta Define o título da aba do navegador e aparece nos resultados do Google. <title>Minha Página</title> Vai dentro do <head>.
<meta>meta Define metadados: charset, viewport, descrição, autor, etc. <meta charset="UTF-8">
<meta name="viewport" content="width=device-width">		 Tag sem fechamento. Essencial para responsividade e encoding.
<link>meta Vincula recursos externos, principalmente arquivos CSS. <link rel="stylesheet" href="style.css"> Também usado para favicons: rel="icon".
<style>meta CSS escrito diretamente no HTML. <style> body { color: red; } </style> Vai no <head>. Prefira arquivos .css externos.
<script>meta Insere ou vincula código JavaScript. <script src="app.js"></script> Coloque antes de </body> para não travar o carregamento.
✏️ Texto e Títulos
TagO que fazExemploObs
<h1> ~ <h6>bloco Títulos de hierarquia. h1 é o mais importante, h6 o menor. <h1>Título Principal</h1>
<h2>Subtítulo</h2>		 Use apenas um <h1> por página. Importante para SEO.
<p>bloco Parágrafo de texto. <p>Este é um parágrafo.</p> Adiciona espaço automático acima e abaixo.
<span>inline Container inline genérico para estilizar partes de um texto. <p>Texto <span style="color:red">vermelho</span></p> Não quebra a linha. Use para estilizar trechos.
<strong>inline Texto em negrito com importância semântica. <strong>Atenção!</strong> Use em vez de <b> pois tem significado semântico.
<em>inline Texto em itálico com ênfase semântica. <em>muito importante</em> Use em vez de <i> para ênfase real.
<br>inline Quebra de linha forçada. Linha 1<br>Linha 2 Tag sem fechamento. Use com moderação.
<hr>bloco Linha horizontal divisória. <hr> Tag sem fechamento. Pode ser estilizada com CSS.
<blockquote>bloco Citação longa em bloco destacado. <blockquote>"Frase famosa"</blockquote> Usa o atributo cite para indicar a fonte.
<q>inline Citação curta inline. Adiciona aspas automaticamente. <q>Citação curta</q> Para citações dentro de um parágrafo.
<code>inline Exibe código com fonte monoespaçada. <code>console.log()</code> Para trechos de código inline.
<pre>bloco Texto pré-formatado — preserva espaços e quebras de linha. <pre> espaços preservados</pre> Combinado com <code> para blocos de código.
<mark>inlineHTML5 Destaca texto como marcador amarelo. <mark>texto destacado</mark> Útil para resultados de busca.
<small>inline Texto menor — notas, avisos legais, copyright. <small>© 2025</small>
<sub>inline Texto subscrito (abaixo da linha base). H<sub>2</sub>O Fórmulas químicas.
<sup>inline Texto sobrescrito (acima da linha base). x<sup>2</sup> Potências e notas de rodapé.
<del>inline Texto deletado — aparece com riscado. <del>R$ 99</del> R$ 49 Útil para preços antigos em promoções.
<ins>inline Texto inserido — aparece sublinhado. <ins>novo conteúdo</ins> Indica conteúdo adicionado.
<abbr>inline Abreviação — mostra o significado no hover. <abbr title="HyperText Markup Language">HTML</abbr> Melhora a acessibilidade.
📋 Listas
TagO que fazExemploObs
<ul>bloco Lista não ordenada (bolinhas). <ul><li>Item</li></ul> Use para itens sem ordem específica.
<ol>bloco Lista ordenada (números). <ol><li>Passo 1</li></ol> Atributos: start, reversed, type.
<li>bloco Item de uma lista (ul ou ol). <li>Meu item</li> Filho direto de <ul> ou <ol>.
<dl>bloco Lista de definições (termos e descrições). <dl><dt>HTML</dt><dd>Linguagem...</dd></dl> Ótimo para glossários.
<dt>bloco Termo da lista de definição. <dt>CSS</dt> Filho de <dl>.
<dd>bloco Descrição do termo na lista de definição. <dd>Cascading Style Sheets</dd> Filho de <dl>.
🔗 Links e Mídia
TagO que fazExemploObs
<a>inline Link — ancora para outra página, seção ou arquivo. <a href="https://google.com" target="_blank">Google</a> target="_blank" abre em nova aba. href="#id" vai pra seção.
<img>inline Exibe uma imagem. <img src="foto.jpg" alt="Descrição da foto"> Sem fechamento. alt é obrigatório para acessibilidade.
<video>blocoHTML5 Incorpora um vídeo. <video src="video.mp4" controls></video> Atributos: controls, autoplay, loop, muted, poster.
<audio>blocoHTML5 Incorpora um áudio. <audio src="musica.mp3" controls></audio> Atributos: controls, autoplay, loop.
<source>meta Define múltiplas fontes para <video> ou <audio>. <source src="video.webm" type="video/webm"> Permite fallback entre formatos.
<iframe>bloco Incorpora outra página ou conteúdo externo (YouTube, Maps). <iframe src="https://youtube.com/embed/..." allowfullscreen></iframe> Cuidado com segurança. Use sandbox quando possível.
<figure>blocoHTML5 Agrupa mídia com legenda. Usado com <figcaption>. <figure><img src="foto.jpg"><figcaption>Legenda</figcaption></figure> Semântico — indica que imagem e legenda são relacionadas.
<figcaption>blocoHTML5 Legenda de uma <figure>. <figcaption>Foto do pôr do sol</figcaption> Filho de <figure>.
<svg>inlineHTML5 Gráfico vetorial escalável diretamente no HTML. <svg width="100" height="100"><circle cx="50" cy="50" r="40"/></svg> Não perde qualidade ao escalar. Ótimo para ícones.
<canvas>blocoHTML5 Área de desenho 2D/3D controlada via JavaScript. <canvas id="meuCanvas" width="400" height="300"></canvas> Usado para jogos, gráficos e animações com JS.
📦 Layout e Containers
TagO que fazExemploObs
<div>bloco Container genérico de bloco. O mais usado para layout. <div class="container">...</div> Sem significado semântico. Use tags semânticas quando possível.
<header>blocoHTML5 Cabeçalho da página ou de uma seção. <header><nav>...</nav></header> Pode haver vários <header> (um por seção).
<nav>blocoHTML5 Bloco de navegação — menus, links principais. <nav><a href="#">Início</a></nav> Usado para leitores de tela identificarem a navegação.
<main>blocoHTML5 Conteúdo principal da página. Único por página. <main>...conteúdo...</main> Melhora acessibilidade e SEO. Apenas um por página.
<section>blocoHTML5 Seção temática do conteúdo. Deve ter um título (h2, h3...). <section id="sobre"><h2>Sobre</h2>...</section> Diferente de <div> — tem significado semântico.
<article>blocoHTML5 Conteúdo independente e reutilizável (post, card, notícia). <article><h2>Post do Blog</h2><p>...</p></article> Pode ser distribuído separado do resto da página.
<aside>blocoHTML5 Conteúdo lateral relacionado — sidebar, nota lateral. <aside><p>Dica relacionada</p></aside> Complementar ao conteúdo principal.
<footer>blocoHTML5 Rodapé da página ou de uma seção. <footer><p>© 2025</p></footer> Pode haver vários <footer> (um por seção).
📊 Tabelas
TagO que fazExemploObs
<table>bloco Define uma tabela. <table>...</table> Não use tabelas para layout — use CSS Grid/Flexbox.
<thead>bloco Agrupa as linhas do cabeçalho da tabela. <thead><tr><th>Nome</th></tr></thead> Semântico — ajuda leitores de tela.
<tbody>bloco Agrupa as linhas do corpo da tabela. <tbody><tr><td>João</td></tr></tbody>
<tfoot>bloco Agrupa as linhas do rodapé da tabela. <tfoot><tr><td>Total</td></tr></tfoot> Pode aparecer antes do <tbody> no código.
<tr>bloco Linha da tabela. <tr><td>Dado</td></tr> Table Row.
<th>bloco Célula de cabeçalho — negrito e centralizado por padrão. <th scope="col">Nome</th> Use o atributo scope para acessibilidade.
<td>bloco Célula de dado da tabela. <td colspan="2">Célula larga</td> colspan = mesclar colunas. rowspan = mesclar linhas.
<caption>bloco Título/legenda da tabela. <caption>Vendas 2025</caption> Primeiro filho de <table>.
📝 Formulários
TagO que fazExemploObs
<form>bloco Container de formulário que envia dados. <form action="/enviar" method="POST">...</form> method: GET (URL) ou POST (corpo da requisição).
<input>inline Campo de entrada — o mais versátil do HTML. <input type="text" name="nome" placeholder="Seu nome">
<input type="email"> <input type="password">
<input type="checkbox"> <input type="radio">
<input type="file"> <input type="submit">
<input type="number" min="0" max="100">
<input type="date"> <input type="color">		 Tag sem fechamento. O atributo type muda o comportamento completamente.
<label>inline Rótulo de um campo. Ao clicar, foca no input associado. <label for="email">E-mail:</label>
<input id="email" type="email">		 O for deve bater com o id do input. Essencial para acessibilidade.
<textarea>bloco Campo de texto multilinha. <textarea rows="5" cols="40" placeholder="Mensagem..."></textarea> Diferente de <input>, tem fechamento.
<select>inline Lista suspensa (dropdown). <select name="estado">
<option value="sp">São Paulo</option>
<option value="rj">Rio de Janeiro</option>
</select>		 Atributo multiple permite seleção múltipla.
<option>inline Opção dentro de um <select> ou <datalist>. <option value="mg" selected>Minas Gerais</option> selected define o padrão.
<button>inline Botão clicável. <button type="submit">Enviar</button>
<button type="button" onclick="minhaFuncao()">Clique</button>		 type="button" não submete o form. type="submit" envia.
<fieldset>bloco Agrupa campos relacionados em um formulário. <fieldset><legend>Dados Pessoais</legend>...</fieldset> Adiciona borda visual ao grupo.
<legend>bloco Título de um <fieldset>. <legend>Endereço</legend> Filho direto de <fieldset>.
<datalist>blocoHTML5 Lista de sugestões para um <input>. <input list="cidades">
<datalist id="cidades">
<option value="São Paulo">
</datalist>		 Autocomplete nativo do browser.
🏷️ Outros Elementos Semânticos
TagO que fazExemploObs
<details>blocoHTML5 Componente abre/fecha (accordion) nativo do browser. <details><summary>Ver mais</summary><p>Conteúdo oculto</p></details> Zero JavaScript necessário.
<summary>inlineHTML5 Título clicável de um <details>. <summary>Clique para expandir</summary> Filho direto de <details>.
<dialog>blocoHTML5 Caixa de diálogo / modal nativa do browser. <dialog id="modal"><p>Olá!</p></dialog> Abrir com JS: document.getElementById('modal').showModal()
<progress>inlineHTML5 Barra de progresso. <progress value="70" max="100"></progress> Sem value = indeterminado (animação).
<meter>inlineHTML5 Medidor de valor em um intervalo (ex: nível de bateria). <meter value="0.7" min="0" max="1">70%</meter> Diferente de <progress> — tem valor atual definido.
<time>inlineHTML5 Representa data ou hora de forma semântica. <time datetime="2025-12-25">Natal 2025</time> Ajuda buscadores a entenderem datas.
<address>bloco Informações de contato do autor ou dono da página. <address>contato@email.com</address> Italicizado por padrão na maioria dos browsers.
<map> + <area>bloco Define áreas clicáveis em uma imagem (image map). <img src="mapa.jpg" usemap="#mapa">
<map name="mapa"><area shape="rect" coords="0,0,100,100" href="pag.html"></map>		 Pouco usado hoje. Preferir CSS + JS.
<template>metaHTML5 Conteúdo HTML inerte que pode ser clonado via JavaScript. <template id="card-template"><div class="card">...</div></template> Não renderiza visualmente. Útil com JS para gerar conteúdo dinâmico.
<noscript>bloco Conteúdo exibido quando o JavaScript está desativado. <noscript><p>Ative o JavaScript.</p></noscript>
