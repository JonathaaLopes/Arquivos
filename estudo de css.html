📦 Box Model
PropriedadeValores comunsO que fazExemplo
widthlayout px, %, vw, auto, min-content, max-content, fit-content Largura do elemento. width: 100%; / width: 300px;
heightlayout px, %, vh, auto Altura do elemento. height: 100vh; / height: auto;
min-width / max-widthlayout px, % Limite mínimo e máximo de largura. max-width: 1200px; margin: 0 auto;
min-height / max-heightlayout px, % Limite mínimo e máximo de altura. min-height: 100vh;
marginlayout px, %, auto — 1 a 4 valores Espaço externo ao elemento. Shorthand: top right bottom left. margin: 20px; / margin: 10px 20px; / margin: 0 auto;
paddinglayout px, % — 1 a 4 valores Espaço interno ao elemento (entre conteúdo e borda). padding: 16px 24px; / padding-top: 10px;
box-sizinglayout content-box (padrão), border-box Define se padding/border incluem no cálculo do width. box-sizing: border-box; /* recomendado */
bordervisual largura estilo cor Borda do elemento. Shorthand para width, style, color. border: 1px solid #ddd; / border-bottom: 2px solid red;
border-radiusvisual px, %, 50% Arredonda os cantos. 50% = círculo. border-radius: 8px; / border-radius: 50%;
outlinevisual largura estilo cor, none Borda fora do layout (não ocupa espaço). Aparece no :focus. outline: none; / outline: 2px solid blue;
overflowlayout visible, hidden, scroll, auto O que fazer com conteúdo que ultrapassa o elemento. overflow: hidden; / overflow-x: auto;
📐 Display e Posicionamento
PropriedadeValores comunsO que fazExemplo
displaylayout block, inline, inline-block, flex, grid, none, contents Define o tipo de renderização do elemento. display: flex; / display: none; / display: grid;
positionlayout static, relative, absolute, fixed, sticky Define o método de posicionamento. static = padrão. position: fixed; top: 0; left: 0; /* navbar fixa */
top / right / bottom / leftlayout px, %, auto Posição do elemento quando position não é static. position: absolute; top: 50%; left: 50%;
z-indexlayout número inteiro (pode ser negativo) Controla a ordem de empilhamento (quem fica na frente). z-index: 100; / z-index: -1;
floatlayout left, right, none Flutua o elemento para um lado. Evite — prefira Flexbox. float: left; width: 50%;
clearlayout left, right, both, none Cancela o efeito de float no elemento. clear: both;
visibilityvisual visible, hidden, collapse Mostra ou esconde sem remover o espaço do elemento. visibility: hidden; /* ocupa espaço mas não aparece */
opacityvisual 0 a 1 Transparência do elemento inteiro (0 = invisível, 1 = opaco). opacity: 0.5; / opacity: 0;
↔️ Flexbox
PropriedadeValores comunsO que fazExemplo
— No elemento PAI (container) —
display: flexlayout flex Ativa o Flexbox no container. Filhos viram flex items. display: flex;
flex-directionlayout row, row-reverse, column, column-reverse Direção do eixo principal. row = horizontal (padrão). flex-direction: column;
justify-contentlayout flex-start, flex-end, center, space-between, space-around, space-evenly Alinha itens no eixo principal (horizontal no row). justify-content: space-between;
align-itemslayout stretch, flex-start, flex-end, center, baseline Alinha itens no eixo cruzado (vertical no row). align-items: center;
align-contentlayout flex-start, flex-end, center, space-between, stretch Alinha múltiplas linhas (quando há wrap). align-content: space-between;
flex-wraplayout nowrap (padrão), wrap, wrap-reverse Permite que itens quebrem para a linha seguinte. flex-wrap: wrap;
gaplayout px, rem — 1 ou 2 valores (row-gap column-gap) Espaço entre os itens. Substitui margin entre itens. gap: 20px; / gap: 16px 24px;
— No elemento FILHO (item) —
flexlayout grow shrink basis — ex: 1, 0 1 auto, 1 1 200px Shorthand para flex-grow, flex-shrink e flex-basis. flex: 1; /* ocupa espaço igual */ / flex: 0 0 300px;
flex-growlayout número (0 = padrão) Fator de crescimento do item quando há espaço sobrando. flex-grow: 1;
flex-shrinklayout número (1 = padrão) Fator de encolhimento quando não há espaço. flex-shrink: 0; /* não encolhe */
flex-basislayout px, %, auto Tamanho inicial do item antes de distribuir o espaço. flex-basis: 200px;
align-selflayout auto, flex-start, flex-end, center, stretch Alinhamento individual do item (sobrescreve align-items). align-self: flex-end;
orderlayout número inteiro (0 = padrão) Altera a ordem visual do item sem mudar o HTML. order: -1; /* vai para o início */
🔲 CSS Grid
PropriedadeValores comunsO que fazExemplo
— No elemento PAI (container) —
display: gridlayout grid Ativa o Grid no container. display: grid;
grid-template-columnslayout px, fr, %, auto, repeat(), minmax() Define as colunas da grade. grid-template-columns: 1fr 1fr 1fr;
grid-template-columns: repeat(3, 1fr);
grid-template-columns: 200px auto 1fr;
grid-template-rowslayout px, fr, auto, repeat() Define as linhas da grade. grid-template-rows: auto 1fr auto;

"header header"
"sidebar main"
"footer footer";
gap / column-gap / row-gaplayout px, rem Espaço entre colunas e/ou linhas da grade. gap: 24px; / column-gap: 20px; row-gap: 16px;
justify-itemslayout start, end, center, stretch Alinha itens dentro das células horizontalmente. justify-items: center;
align-itemslayout start, end, center, stretch Alinha itens dentro das células verticalmente. align-items: center;
— No elemento FILHO (item) —
grid-columnlayout início / fim — ex: 1 / 3, 1 / -1, span 2 Quantas colunas o item ocupa. grid-column: 1 / 3; / grid-column: span 2;
grid-rowlayout início / fim Quantas linhas o item ocupa. grid-row: 1 / 3;
grid-arealayout nome definido em grid-template-areas Posiciona o item em uma área nomeada. grid-area: header;
justify-self / align-selflayout start, end, center, stretch Alinhamento individual do item dentro da célula. justify-self: end;

🔤 Tipografia
PropriedadeValores comunsO que fazExemplo

font-sizetexto px, rem, em, %, vw Tamanho do texto. rem = relativo ao root (html). font-size: 1rem; / font-size: 16px; / font-size: clamp(14px, 2vw, 20px);
font-weighttexto 100–900, normal (400), bold (700) Espessura da fonte. font-weight: 700; / font-weight: bold;
font-styletexto normal, italic, oblique Estilo da fonte. font-style: italic;
line-heighttexto número, px, % (1.5 recomendado) Espaçamento entre linhas. Sem unidade é relativo ao font-size. line-height: 1.6; / line-height: 24px;
letter-spacingtexto px, em Espaço entre letras (tracking). letter-spacing: 0.1em; / letter-spacing: 2px;
word-spacingtexto px, em Espaço entre palavras. word-spacing: 4px;
text-aligntexto left, right, center, justify Alinhamento horizontal do texto. text-align: center; / text-align: justify;
text-decorationtexto none, underline, line-through, overline Decoração do texto (sublinhado, riscado...). text-decoration: none; / text-decoration: underline wavy red;
text-transformtexto uppercase, lowercase, capitalize, none Transforma maiúsculas/minúsculas sem alterar o HTML. text-transform: uppercase; / text-transform: capitalize;
text-shadowtexto x y blur cor Sombra no texto. text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
text-overflowtexto clip, ellipsis O que mostrar quando o texto excede a largura. Use com overflow: hidden. white-space: nowrap; overflow: hidden; text-overflow: ellipsis;

word-breaktexto normal, break-all, keep-all Se palavras longas podem quebrar no meio. word-break: break-all;
@font-facetexto font-family, src Carrega uma fonte personalizada. @font-face { font-family: 'Minha'; src: url('font.woff2'); }

scale(x, y)animação número (1 = normal) Escala o elemento. scaleX(), scaleY(). transform: scale(1.1); /* 10% maior no hover */
rotate(ângulo)animação deg, rad, turn Gira o elemento. transform: rotate(45deg); / transform: rotate(-90deg);
skew(x, y)animação deg Inclina o elemento. transform: skewX(20deg);

perspectiveanimação px Profundidade da perspectiva 3D. perspective: 1000px;
rotateX / rotateY / rotateZanimação deg Rotação em 3D nos eixos X, Y e Z. transform: rotateY(45deg);

transition-timing-functionanimação ease, linear, ease-in, ease-out, ease-in-out, cubic-bezier() Curva de velocidade da transição. transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);

0% { transform: scale(1); }
50% { transform: scale(1.1); }
100% { transform: scale(1); }
}
animation-iteration-countanimação número, infinite Quantas vezes a animação repete. animation-iteration-count: infinite;

animation-fill-modeanimação none, forwards, backwards, both Estado do elemento antes/depois da animação. animation-fill-mode: forwards;
animation-play-stateanimação running, paused Pausa ou retoma a animação. animation-play-state: paused; /* pausa no hover */
📱 Responsividade e Variáveis
RecursoSintaxeO que fazExemplo
@mediaresponsivo min-width, max-width, orientation, print Aplica CSS apenas em determinadas condições (tamanho de tela). @media (max-width: 768px) {
.menu { display: none; }
}
@media (min-width: 1024px) { ... }
Breakpoints comunsresponsivo — Referência de tamanhos de tela padrão. /* Mobile */ max-width: 480px
/* Tablet */ max-width: 768px
/* Desktop */ max-width: 1024px
/* Wide */ min-width: 1200px
--variável (Custom Properties)responsivo qualquer valor CSS Variável CSS reutilizável. Declarada com -- e usada com var(). :root { --cor-primaria: #2E7D32; }
button { background: var(--cor-primaria); }
clamp()responsivo mín, preferido, máx Valor fluido entre um mínimo e máximo. Ideal para font-size responsivo. font-size: clamp(14px, 2.5vw, 22px);
Unidades relativasresponsivo vw, vh, vmin, vmax, rem, em, % Unidades que se adaptam ao tamanho da tela ou do pai. width: 100vw; /* 100% da viewport */
font-size: 1rem; /* relativo ao :root */
padding: 1em; /* relativo ao font-size do pai */
@media (prefers-color-scheme)responsivo dark, light Detecta se o usuário usa dark mode ou light mode. @media (prefers-color-scheme: dark) {
body { background: #000; color: #fff; }
}
@media printresponsivo print CSS aplicado ao imprimir a página. @media print { nav, footer { display: none; } }
object-fitlayout fill, contain, cover, none, scale-down Como imagem/vídeo preenche o container. img { width: 100%; height: 200px; object-fit: cover; }
object-positionlayout center, top, left, px, % Posiciona a imagem dentro do container com object-fit. object-position: center top;
aspect-ratiolayout 16/9, 4/3, 1, auto Mantém proporção do elemento automaticamente. aspect-ratio: 16 / 9;
cursorvisual pointer, default, text, move, not-allowed, grab... Aparência do cursor do mouse sobre o elemento. cursor: pointer; / cursor: not-allowed;
pointer-eventsvisual auto, none Desativa interações do mouse no elemento. pointer-events: none; /* cliques passam pelo elemento */
user-selectvisual auto, none, all, text Controla se o texto pode ser selecionado pelo usuário. user-select: none; /* impede seleção */
resizelayout none, both, horizontal, vertical Se o usuário pode redimensionar o elemento (textarea). textarea { resize: vertical; }
scroll-behaviorlayout auto, smooth Animação de scroll ao navegar para âncoras (#). html { scroll-behavior: smooth; }
